# Getting Started with Teramot

Welcome to Teramot! This guide will walk you through everything you need to connect your data, configure your environment, and start using AI-powered analytics.

## What is Teramot?

Teramot is an AI-powered data engineering platform that enables organizations to build intelligent data pipelines using a medallion architecture (Bronze → Silver → Gold layers). Our platform lets you:

- Connect any data source (databases, APIs, files)
- Automatically generate and manage ETL pipelines
- Create AI-driven analytical tables (Gold Layer) with natural language
- Query your data via the MCP API

## Prerequisites

Before getting started, ensure you have:

- A Teramot account (contact [support@teramot.com](mailto:support@teramot.com))
- Access to your data sources (database credentials, API keys, etc.)
- An AWS account (for multi-tenant deployments)
